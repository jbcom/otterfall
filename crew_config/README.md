# CrewAI Development Setup

## Usage

```bash
# Install dependencies
cd python/crew_agents && uv run crewai install

# Run full crew
cd python/crew_agents && uv run crewai run

# Or via process-compose
process-compose up rivermarsh_crew
```

## Architecture

**CrewBase Native DSL** - All configuration in `python/crew_agents/crewbase.yaml`:
- 10 specialized agents (Technical Director as hierarchical manager)
- Alpha→Work→Omega workflow pattern
- Native MCP integration (no custom adapters)
- Role-based tool filtering using `mcp://server/tool` syntax

## Configuration

- `python/crew_agents/crewbase.yaml` - **Single source of truth**
- `.mcp.json` (root) - MCP servers (generated by `ruler apply --nested`)

Legacy YAML files (`crew_config/*.yaml`) are kept for reference but no longer used by the crew.
